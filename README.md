# Durable Archives Management Solution API

Static Swagger UI documentation for the Durable Archives Management Solution API.

When GitHub Pages has deployed, the public documentation is available at:

https://dc-durable.github.io/archives-management-solution/

## Local Preview

Serve the repository with any static file server and open the root URL. For example:

```sh
python3 -m http.server 8000
```

Then visit http://localhost:8000/.
